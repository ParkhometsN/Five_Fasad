1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты информации о физических лицах (далее — «Пользователи»), которая может быть получена Обществом с ограниченной ответственностью «Пятый фасад» (далее — «Компания», «Мы») при использовании Пользователем услуг, предоставляемых Компанией, включая, но не ограничиваясь: заказ услуг монтажа натяжных потолков, посещение сайта [вставить адрес сайта], а также взаимодействие с Компанией по телефону, электронной почте или через формы обратной связи.

1.2. Используя услуги Компании и предоставляя ей свои персональные данные, Пользователь выражает свое согласие с условиями данной Политики.

1.3. Обработка персональных данных осуществляется в соответствии с законодательством Российской Федерации, в частности, Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

2. Состав собираемой информации

2.1. Компания может собирать следующие категории персональных данных:

Данные для связи: Фамилия, имя, отчество; номер телефона; адрес электронной почты (e-mail).
Данные для оказания услуг: Адрес объекта, где требуется монтаж натяжного потолка; параметры помещения (площадь, высота, форма); предпочтения по типу потолка, цвету, производителю; информация, необходимая для составления сметы и договора.
Техническая информация: При посещении сайта автоматически собираются данные об IP-адресе, типе браузера, времени доступа, адресе запрашиваемой страницы. Эта информация используется для анализа производительности сайта и улучшения его работы.
3. Цели сбора и обработки персональных данных

3.1. Компания использует персональные данные Пользователей в следующих целях:

Для связи с Пользователем для обработки его заявки, консультации по услугам, согласования деталей заказа.
Для заключения и исполнения договора на оказание услуг монтажа натяжных потолков.
Для расчета стоимости услуг, формирования коммерческого предложения и сметы.
Для отправки информационных сообщений, связанных с исполнением заказа (например, уведомление о готовности, вызов замерщика, согласование даты монтажа).
Для направления рекламных и информационных материалов (только с согласия Пользователя).
Для улучшения качества обслуживания и развития наших услуг.
4. Условия обработки и передачи персональных данных

4.1. Компания обязуется не разглашать персональные данные Пользователей и не передавать их третьим лицам, за исключением следующих случаев:

Согласие Пользователя: Если Пользователь явно выразил свое согласие на такие действия.
Для исполнения договора: Передача данных субподрядчикам и партнерам, непосредственно участвующим в оказании услуг (например, курьерским службам для доставки материалов, при необходимости). Указанные лица имеют право обрабатывать персональные данные только в целях исполнения своих обязательств перед Компанией.
По требованию закона: В случаях, когда раскрытие информации требуется в соответствии с действующим законодательством РФ по законному запросу уполномоченных государственных органов.
4.2. Компания принимает все необходимые организационные и технические меры для защиты персональных данных Пользователя от неправомерного или случайного доступа, уничтожения, изменения, блокирования, копирования, распространения, а также от иных неправомерных действий.
